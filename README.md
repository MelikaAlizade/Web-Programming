## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge)
- [Python 3.x](https://www.python.org/) (for backend assignments)
- [Node.js](https://nodejs.org/) (for TypeScript/JS tooling)

### Running Frontend Assignments

Open any `.html` file directly in your browser, or use a local dev server:

```bash
# Using Python's built-in HTTP server
python -m http.server 8000
# Then visit http://localhost:8000
```

### Running Backend Assignments

```bash
cd hw3   # or whichever folder has Python code
pip install -r requirements.txt   # if applicable
python app.py
```
